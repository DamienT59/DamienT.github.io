Excel_Finder

Une application permettant à son utilisateur d'obtenir un nouveau fichier excel regroupant toutes les informations identiques entre 2 fichiers excels au format csv séparé par des points virgules.

Ce logiciel vous demande la colonne a comparer dans les 2 fichiers, si vous cochez "Première ligne ?" le logiciel va alors créer sur votre nouveau fichier une ligne regroupant les informations ( exemple: Nom,Prénom,Date,etc...)

Pour vous aider, vous avez le bouton d'aide "?"
